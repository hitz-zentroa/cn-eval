How to compute repetition rate on a text, which is a txt file containing one CN per line. 
For computing more accurate repetition rate, the text should be randomly shuffled.
The repetition rate copmuted will be shown on the shell.

1. Run: !chmod 755 RR -R
2. specify the filename to be computed in run_repetition_rate.sh
3. Run: bash ./run_repetition_rate.sh
